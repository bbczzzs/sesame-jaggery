// turbo-all

## Deploy to GitHub

1. Stage all changes
```bash
git add .
```

2. Commit changes
```bash
git commit -m "Update project"
```

3. Push to GitHub
```bash
git push
```

## Deploy to Vercel

4. Deploy to production
```bash
vercel --prod
```
